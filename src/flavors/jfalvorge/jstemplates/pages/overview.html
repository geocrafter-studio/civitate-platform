Bem vindo ao Civitate!

Versão demonstração para a freguesia do {{ config.app.freguesia }}

Civitate é um projecto da Geocrafter - Geospatial Studio.

O Civitate é simples e intuitivo, basta seguir alguns passos:

  • Para criar uma nova sugestão/ocorrência basta clicar em {{ config.place.add_button_label }} e arrastar o mapa para o local desejado.
  • Escolha o tipo de sugestão/ocorrência que pretende introduzir.
  • Preencha o formulário com os dados obrigatórios.
  • Submeta a sugestão/ocorrência e...é tudo! O seu novo local já deverá estar visível no mapa.
  • Pode utilizar a autenticação pelas principais redes sociais para partilhar mais facilmente a sua sugestão/ocorrência.
  • Pode criar comentários em sugestões/ocorrências já existentes.
  • Pode apoiar sugestões/ocorrências já existentes, tal como acontece nas redes sociais.

Estou pronto!
